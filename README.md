# fakeTec v4.1
FakeTec V4.XX

Robinson Labs have jumped on board FakeTec starting @ V4 from lupus worax. This is a complete redesign of the board correcting many issues. Robinson Labs will keep our own version files here so anyone that wants to make these boards will be able to.

WHATS NEW IN V4.1

1, PROPER POWER PATH

+3.3v now has a proper power path, voltage no longer dipps when radio keys up.

2, lAYOUT CORRECTED

Silk screen has been redone from scratch.

3, BATTERY CONTACTS

Battery contacts have been added so users have a native place to solder a protected lipo.

4, GPS PINS ADDED

Pin header now fitted for the gps.

5, PROMICRO HOLES REMOVED

Through holes for the ProMicro NRF have been removed, you can now solder both modules smt on the hot plate in one heating!

6, RESISTOR NETWORK MOVED

Moved the battery % resistors back to the bottom where they should be.

7, REDESIGN

Inserted proper pin headers for voltage handeling mosfets

RTG1 now controls the ground pin on the GPS header.
RTG2/RGT3 (buzzer/vibro) now have there own pins which are switched also now.
